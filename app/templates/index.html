Under the hood

This recommendation system feeds on browsing information such as a) the time you spend visiting websites, b) the number of times you visit them, c) the hours you visit them and d) your geographical location. It then group people with similar interests together and provides you suggestions about the websites you may like, considering the one you already liked in the past. You can read more about this project on my weblog

To do so, I'm mainly using machine learning algorithms and text retrieval techniques. Here is the set of tools I'm using: The Flask framework to serve those pages (assisted by haproxy, nginx and gunicorn), the awesome jetpack sdk to make firefox extensions, the MongoDB database to store data and the scikits.learn to extract text features and compute recommendations
